### The project folder must have include folder!

- folder structure 

folder structrue should look similar to this 

```
.
├── LICENSE
├── README.md
├── libs
│   └── [put your custom lib or others]
│       └── [demo.h]
└── main_template
    ├── Makefile
    ├── include(This folder must be in project folder)
    └── src
        └── main.c
```
